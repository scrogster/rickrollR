# rickrollR

R package providing some functions for [rickrolling](https://en.wikipedia.org/wiki/Rickrolling) in R.

## Installation

```
require(devtools)
install_github("scrogster/rickrollR")
```

## Rickrolling in the console

```
library(rickroll)
rickroll_console()
```

## Rickrolling on youtube

```
library(rickroll)
rickroll_youtube()
